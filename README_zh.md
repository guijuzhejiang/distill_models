# 知识蒸馏模型压缩项目

English | [中文](README_zh.md)

本项目展示了如何使用 **知识蒸馏 (Knowledge Distillation, KD)** 技术，利用强大的教师模型 (Vision Transformer - ViT) 来指导轻量化学生模型 (MobileNetV2) 的训练。

## 🚀 项目概览

知识蒸馏的核心思想是将“知识”从一个大型、高性能的教师模型迁移到一个更小、更高效的学生模型中。通过学习教师模型输出的“软标签”（概率分布），学生模型能够获得比直接从零训练更好的性能。

### 🧪 实验设置

- **数据集**: `beans` (图像分类)
- **教师模型**: `merve/beans-vit-224` (Vision Transformer)
- **学生模型**: `MobileNetV2` (从零初始化)
- **损失函数**: 软标签的 KL 散度 + 硬标签的交叉熵

## 📊 性能对比

| 训练方式 | 模型 | 测试集准确率 |
| :--- | :--- | :--- |
| 从零开始训练 (Baseline) | MobileNet | 63% |
| **知识蒸馏训练** | **MobileNetV2** | **72%** |

如上表所示，通过知识蒸馏技术，学生模型的准确率提升了 **9%**，在保持轻量化架构的同时，极大缩小了与大型模型之间的性能差距。

## 💡 知识蒸馏的优点

为什么在实际生产中推荐使用知识蒸馏？

1.  **模型压缩**: 显著减少模型参数量和体积，便于在手机、嵌入式设备等端侧部署。
2.  **推理加速**: 小模型具有更低的延迟，能够支持实时推理应用。
3.  **性能增强**: 允许小模型学习教师模型发现的类间关系（暗知识），从而获得超越常规训练的泛化能力。
4.  **知识迁移**: 软标签包含比硬标签（0/1）更丰富的信息量，引导学生模型更平稳地收敛。
